Footnote, BrickHack, 2019
Foonote is a Google Chrome plugin that allows you to comment on any page and see other people's comments too. It was by far one of the most challenging projects that I worked on. We setup a Java server from the ground up to handle commenting and hooked it into an SQL server to create automatic e-mail verification for users to create a unique account that would remember their comments. Unfortunately, it requires a constant server to be up, and we don't have it up right now - but I've left the installation link here.
